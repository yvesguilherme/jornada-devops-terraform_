## How to use

1. Initialize Terraform:
```
terraform init
```

2. Set the required variables in the `terraform.tfvars` file:
```
variable = "valor"
```

3. Preview the changes that will be made:
```
terraform plan
```

4. Apply the changes:
```
terraform apply
```
